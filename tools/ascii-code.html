ASCII Code Generator

Convert text to ASCII codes and vice versa. Enter text to get its ASCII codes, or enter ASCII codes (space-separated numbers) to convert back to text.

ASCII codes are space-separated numbers representing each character.

Click the copy button to copy the result to your clipboard.
