Number System Converter

Convert numbers between different number systems (Binary, Decimal, Hexadecimal, Octal, etc.).

Enter a number in the selected base system (e.g., "FF" for hexadecimal, "1010" for binary).

The result will be shown with the appropriate prefix (0b for binary, 0o for octal, 0x for hexadecimal).

Click the copy button to copy the result to your clipboard.
